# django-vuejs
Django + Vue.js で簡単なWEBアプリケーションを作成する

# 実行方法
## 1. リポジトリのクローン
```bash

```
***
## 2. バックエンド実行
### 2-1. パッケージのインストール
```bash
cd server
pip install -r requirements.txt
```
### 2-2. マイグレート
```bash
python manage.py migrate
```
### 2-3. 開発サーバ起動
```bash
python manage.py runserver
```
***
## 3. フロントエンド実行

### 3-1. セットアップ
```bash
npm install
```
### 3-2. 実行
```basu
npm run serve
```
